1. добавляем папку "memes for Mr.Nesterchuk" в локальный репозиторий
2. git add "memes for Mr.Nesterchuk" (добавляем папку "memes for Mr.Nesterchuk" к коммиту)
3. git commit -m "Название коммита" (сохранение изменений в локальном репозитории) 
4. git push origin master (отправка в удаленный репозиторий) 

Все выполняется при условии, что удаленный репозиторий ранее был подключен
